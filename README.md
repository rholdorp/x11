# X11 - Team allocation tooling

Install the application:

```sh
npm install
```

Run dev:

```sh
npm run dev
```

Run build:

```sh
npm run build
```

Install prisma packages:

```sh
npm install --save-dev prisma
npm install @prisma/client
```

initialize prisma with sqlite:

```sh
npx prisma init --datasource-provider sqlite
```
