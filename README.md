# X11 - Team allocation tooling

To install x11:

1. Clone the code base.

2. Install the application and dependencies:

```sh
npm install
```

3. Install prisma packages:

```sh
npm install --save-dev prisma
npm install @prisma/client
```

4. Push the database:

```sh
npx prisma db push
```

5.  Run dev:

```sh
npm run dev
```
